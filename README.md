# job-recommendation-system
Demo hệ thống tư vấn vị trí việc làm cho sinh viên ngành Quản lý thông tin USSH VNU-HCM


Quy trình tìm kiếm
01
Nhập từ khóa tìm kiếm
Người dùng nhập tên vị trí việc làm tại ô tìm kiếm trên trang chủ.

02
Hệ thống xử lý truy vấn
Chuẩn hóa từ khóa và đối chiếu với dữ liệu tuyển dụng đã lưu trữ.

03
Trả về danh sách kết quả
Hiển thị các vị trí phù hợp để người dùng chọn xem chi tiết hoặc tiếp tục đối sánh hồ sơ cá nhân.
